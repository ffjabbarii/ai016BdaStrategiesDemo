Python BlueprintAPI

FastAPI + AWS Blueprint API Implementation

Python BlueprintAPI Project

This implementation uses FastAPI and AWS Blueprint API to provide template-based document processing for W-2 forms and bank statements.

Architecture


Client Request → FastAPI → AWS Blueprint API → Template Processing → Structured Response

Key Features

  • FastAPI Framework: High-performance async REST API
  • Blueprint Templates: Pre-configured document templates
  • Automatic Validation: Built-in field validation and confidence scoring
  • Lambda Deployment: Serverless architecture with API Gateway

Project Structure


python/BlueprintAPI/

API Endpoints


POST /process/w2                  # Process W-2 document
POST /process/bank-statement      # Process bank statement  
GET  /health                      # Health check endpoint

Local Development


# Install dependencies
pip install -r requirements.txt

# Run locally
python src/api.py

# Test endpoints
curl -X POST "http://localhost:8000/process/w2" \
     -H "Content-Type: multipart/form-data" \
     -F "file=@sample_w2.pdf"

Deployment


# Configure AWS credentials
aws configure

# Deploy to Lambda
python deploy.py

# The deployment creates:
# - Lambda function: blueprint-api-processor
# - API Gateway: REST API with endpoints
# - IAM roles: Lambda execution role with Textract permissions

Sample Usage


import requests

# Process W-2 document
with open('sample_w2.pdf', 'rb') as f:
    response = requests.post(
        'https://api-gateway-url/process/w2',
        files={'file': f}
    )

result = response.json()
print(f"Employee SSN: {result['extracted_data']['employee_info']['ssn']}")
print(f"Wages: {result['extracted_data']['tax_info']['wages']}")
