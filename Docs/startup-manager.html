Available Commands

# Start Python W-2 processor on default port
python project_manager.py python_blueprint_api backend start

# Start C# analyzer on custom port
python project_manager.py csharp_analyze_document backend start 5555

# Stop specific project
python project_manager.py python_blueprint_api backend stop 8000

# Start documentation portal
python project_manager.py documentation_portal frontend start 8080

ğŸ†˜ Troubleshooting

Common Issues and Solutions:

  • GUI won't open: Install tkinter: python -m tkinter
  • Dependencies missing: Run pip install requests psutil
  • Port already in use: Change port number in GUI or stop conflicting process
  • Service won't start: Check project path exists and dependencies installed
  • Browser shows error: Wait 10-15 seconds for service to fully start
  • Health check fails: Verify service is running and endpoint is correct
