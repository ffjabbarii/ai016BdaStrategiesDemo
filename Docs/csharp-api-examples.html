C# BDA BLUEPRINT

1. Create C# Blueprint Project

Purpose: Create AWS Blueprint project using C# implementation

Request:

curl -X 'POST' \ "http://localhost:5000/blueprint/create" \ -H 'accept: application/json' \ --data-urlencode "projectName=csharp-demo-project" \ --data-urlencode "documentType=invoice" \ --data-urlencode "description=C# Demo Invoice Analysis Project"

Response:

{
  "status": "success",
  "projectArn": "arn:aws:textract:us-east-1:blueprint:project/csharp-demo-project",
  "s3Bucket": "bda-blueprint-csharp-demo-project-1765760000",
  "adapterId": null,
  "projectName": "csharp-demo-project",
  "documentType": "invoice",
  "message": "C# Blueprint project created in your AWS account",
  "language": "C#"
}
