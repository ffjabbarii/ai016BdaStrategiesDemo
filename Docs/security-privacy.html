Checklist for financial docs

Storage

  • KMS encryption
  • Separate raw vs derived outputs
  • Lifecycle retention policies

Access control

  • Least privilege roles
  • Prefer SSO/short-lived creds
  • Restrict raw PDF downloads

Audit

  • Audit logging
  • Alert on unusual access

Don’t do this

  • Don’t commit real PDFs to git
  • Don’t store full SSNs unless required
