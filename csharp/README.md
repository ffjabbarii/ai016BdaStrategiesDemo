# C# Implementation

This directory contains the C# .NET implementations of all three BDA patterns using AWS services.

## Projects
- **BlueprintAPI/** - AWS Blueprint API with ASP.NET Core
- **Textract/** - Direct Textract API with AWS Lambda
- **AnalyzeDocument/** - Enhanced Textract SDK with advanced features

## Requirements
- .NET 6.0 or later
- AWS CLI configured
- Visual Studio or VS Code

## Quick Start
Navigate to any project directory and run:
```bash
dotnet restore
dotnet run
```
