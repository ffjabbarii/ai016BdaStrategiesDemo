# Python Implementation

This directory contains the Python implementations of all three BDA patterns using AWS services.

## Projects
- **BlueprintAPI/** - AWS Blueprint API with FastAPI
- **Textract/** - Direct Textract API with Lambda
- **AnalyzeDocument/** - Enhanced Textract SDK with advanced features

## Requirements
- Python 3.8+
- AWS CLI configured
- pip for package management

## Quick Start
Navigate to any project directory and run:
```bash
pip install -r requirements.txt
python src/api.py
```
